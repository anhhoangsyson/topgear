# Admin: Quản lý Đơn hàng

Vị trí: `src/app/admin/(otherPages)/orders/`

Mô tả
- Xem danh sách đơn, chi tiết đơn, thay đổi trạng thái (ví dụ: đang xử lý, đã giao), quản lý hoàn tiền.

Nội dung chính
- Order table, filter theo trạng thái, chi tiết order với thông tin khách hàng, items và thông tin vận chuyển.

Lưu ý
- Các thao tác cập nhật trạng thái sẽ gọi API backend và cần quyền admin.
